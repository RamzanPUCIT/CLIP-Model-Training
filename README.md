# CLIP Model Training

## 📌 Overview
This project demonstrates a simple implementation of CLIP (Contrastive Language Image Pretraining).

## 🧠 Concept
CLIP is a model that learns the relationship between images and text using contrastive learning.

## 🚀 What I Did
- Trained a CLIP model
- Learned image-text similarity
- Evaluated model performance using loss

## ⚙️ Tools Used
- Python
- PyTorch
- Google Colab

## 📊 Results
- Training loss decreased
- Validation loss improved
- Model trained successfully

## 📁 Project Structure
- notebook.ipynb → main training notebook
- requirements.txt → dependencies
